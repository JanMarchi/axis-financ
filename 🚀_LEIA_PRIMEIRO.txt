🔧 INSTALAR POSTGRESQL E REDIS (WSL)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Abra Windows Terminal (ou PowerShell)

2. Digite:
   wsl

3. Você entrou no WSL. Agora execute:
   sudo apt update && sudo apt install postgresql-16 redis-server

4. Inicie os serviços:
   sudo service postgresql start
   sudo service redis-server start

5. Verifique:
   psql -U postgres -c "\q"
   redis-cli ping

Se funcionar, pode fechar o WSL e voltar para o PASSO 2 acima.
